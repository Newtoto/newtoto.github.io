Squid Vs Eel (2016)

Description

Squid VS Eel is a 2 player couch competitive game programmed in python using pygame.

It is played by sharing a single keyboard with each player taking on the role of either the squid or the eel, each possessing a set of unique abilities designed to inflict damage, or counter the opponent's attacks.

The use of cooldowns introduces an element of timing and skill.

My role and contribution

The team consisted of 4 programmers, including myself.

For this game I was involved with both programming along with the 3 others, as well as making all the artwork and procedurally generated music used in the game.

Hyper Heatwave Runner (Global Game Jam 2017)

Description

Hyper Heatwave Runner is an endless runner game made in Unity.

It is set from the perspective of a brit abroad in France trying to cool down in the heatwave by picking up chilling drinks and ice creams and avoiding hot chilli peppers. All this while a load of baguettes fly at your face.

My role and contribution

The team of 7 for this project consisted of 3 programmers, 2 artists, and 2 designers.

I was involved with the programming, as well as making the music and sound effects for the game.

Global Game Jam Github repository

Descension (2017)

Description

Descension is a first person melee hoard survival game made in UE4.

It is set in the underworld and created by Demon Crypt. Fight your way through masses of disturbing creatures with a hammer or dual axes in order to unleash as much havoc as you can, before your inevitable death that is.

My role and contribution

The Demon Crypt team consisted of 11 people, the breakdown of which is on the website linked below.

I was one of 3 programmers on the team.

Crossbeam (2017)

Description

Crossbeam is a 2D puzzle game made in UE4 where you shoot 4 turrets to around obstacles in order to hit targets.

This game is designed with a custom controller in mind which was made using a button, 2 potentiometers and an Arduino. You can see the controller in action here.

My role and contribution

I was the only person working on this project.
