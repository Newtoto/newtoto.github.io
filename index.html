  • Squid Vs Eel

    Squid Vs Eel is a 2 player couch competitive game in which players adopt the roles of either a squid or eel, each with their own ability set designed around counter attack tactics. This is a work in progress so functionality and assets are still to be built in. I am the scrum master for this project and am currently working with 3 others to make this game. All members of the team, including myself, have contributed to the coding, the github repository of which is available below. The assets are all my own creation.
